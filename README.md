## Campers

Клієнтський застосунок для каталогу кемперів на React + Vite.

## Можливості

- перегляд списку кемперів
- фільтрація та пошук
- додавання до обраного
- перегляд деталей кемпера

## Технології

- React
- TypeScript
- Vite
- Redux Toolkit
- React Router
- Axios

## Встановлення та запуск

1. Встановити залежності:

```bash
npm install
```

2. Запустити режим розробки:

```bash
npm run dev
```

3. Збірка для продакшну:

```bash
npm run build
```

4. Перегляд зібраного проєкту:

```bash
npm run preview
```

## Корисні скрипти

- `npm run dev` — локальний сервер розробки
- `npm run build` — продакшн-збірка
- `npm run lint` — перевірка стилю коду
- `npm run preview` — попередній перегляд збірки
- `npm run generate:icons` — генерація типів для іконок

## Структура проєкту

- `src/` — основний код застосунку
- `public/` — статичні файли
- `scripts/` — службові скрипти
